== Kubernetes Secrets

This example shows how to use Kubernetes secrets for configuration properties in a Camel route.

The route exposes a REST endpoint that returns a news headline read from a Kubernetes secret.

=== How to run

First, deploy to Kubernetes:

    camel kubernetes run news-service.camel.yaml application.properties

The secret `news.headline` is mounted from `/etc/camel/secrets/` as configured in `application.properties`.
